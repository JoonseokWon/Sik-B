# Food-Fee를 계산하는 간단한 툴

엔터산업 정산 과정에서 식비처럼 작지만 자주 발생하는 비용을 빠르게 분배하고, 사람 손이 필요한 예외 건만 따로 표시하는 로컬 웹 도구입니다.

## 실행 방법

가장 쉬운 방법은 `Sik-B.exe`를 더블 클릭하는 것입니다.

1. 프로젝트 폴더에서 `Sik-B.exe`를 더블 클릭합니다.
2. 브라우저가 열리면 `samildol 더미 연동` 버튼으로 가상 출퇴근 기록과 일정표 데이터를 불러옵니다.
3. 멤버, 식비 결제 건, 정산 제외 인원을 수정합니다.
4. `Excel 내보내기` 버튼을 누르면 Excel에서 열 수 있는 CSV 파일이 다운로드됩니다.

별도의 서버나 설치 과정 없이 `index.html`을 직접 열어도 사용할 수 있습니다.

## 개요

Food-Fee는 배달 음식, 도시락, 간식처럼 여러 명이 함께 사용한 비용을 멤버별 정산금에 반영하기 위한 보조 툴입니다.

핵심 방향은 1원까지 완벽하게 추적하는 것이 아니라, 사소한 비용 처리에 들어가는 인적자원을 줄이는 것입니다. 기본은 참석자 기준 균등 배분으로 처리하고, 금액이 큰 식사만 승인 필요 건으로 분리합니다.

## 주요 정책

- 식비 결제 건은 실제 식사 인원을 기준으로 입력합니다.
- 매니저처럼 고정급을 받고 식비 제공 대상인 인원은 `정산 제외`에 넣습니다.
- 정산 제외 인원은 실제로 같이 먹었더라도 멤버 정산 공제 대상에서 빠집니다.
- 남은 정산 대상 인원에게만 비용을 균등 배분합니다.
- 1인 식사 금액이 30,000원을 초과하면 자동으로 `승인 필요`로 표시합니다.
- 결과는 Excel에서 바로 열 수 있는 CSV 파일로 내보낼 수 있습니다.

예: 배민 55,000원을 Ari, Jin, Mina, Manager Kim이 먹었고 Manager Kim을 정산 제외하면, 55,000원은 Ari, Jin, Mina 3명에게만 나뉩니다. 1인 금액은 18,334원이며 승인 기준 30,000원 이하라 자동 처리됩니다.

## 주요 기능

- 그룹명, 정산 기간, 1인 식사 승인 기준 입력
- 멤버별 매출액과 정산 비율 입력
- 식비 결제 건별 금액, 식사 인원, 정산 제외 인원 입력
- 매니저 제외 후 균등 배분 자동 계산
- 1인 30,000원 초과 식사 자동 예외 처리
- 멤버별 식비 공제액과 공제 후 정산금 확인
- 멤버별 정산 결과와 비용별 분배 결과 CSV 내보내기
- `samildol` 가상 그룹의 출퇴근 기록과 캘린더 일정표 더미 데이터 관리

## 더미 연동 방식

현재 버전은 실제 출석체크 앱이나 캘린더 API와 직접 연결되어 있지는 않습니다. 대신 `samildol`이라는 가상 그룹 데이터를 사용해, 외부 출퇴근 기록표와 캘린더 일정표에서 데이터를 가져온 상황을 시뮬레이션합니다.

- 출퇴근 기록: 날짜, 멤버, 출근 상태
- 일정표: 날짜, 일정명, 참여 멤버
- 식비 결제 건: 해당 일정 또는 출근일에 발생한 비용, 식사 인원, 정산 제외 인원

## 현재 계산식

- 멤버별 기본 정산금 = 멤버 매출액 x 정산 비율
- 정산 대상 인원 = 식사 인원 - 정산 제외 인원
- 1인 식비 = 결제 금액 / 정산 대상 인원, 원 단위 올림
- 멤버별 식비 공제액 = 해당 멤버가 정산 대상에 포함된 식비 건의 1인 식비 합계
- 공제 후 정산금 = 기본 정산금 - 식비 공제액
- 승인 필요 여부 = 1인 식비가 승인 기준을 초과하는지 여부

## 확인이 필요한 질문

- 매니저 몫은 회사 비용으로 별도 기록해야 하나요, 아니면 총액 전체를 멤버에게 나누나요?
- 30,000원 기준은 세금/배달비 포함 총액 기준인가요, 음식값만 기준인가요?
- 승인 필요 건도 일단 정산금에서 차감하나요, 승인 전까지 보류하나요?
- 1인 금액을 원 단위로 올림 처리하는 것이 맞나요, 10원 또는 100원 단위 반올림이 필요한가요?
- 스태프, 게스트, 외부 협력자는 매니저와 같은 정산 제외 인원으로 처리하나요?

## 파일 구성

- `Sik-B.exe`: 원클릭 실행 파일
- `_start.bat`: 보조 실행 파일
- `index.html`: 웹 화면
- `styles.css`: 화면 디자인
- `app.js`: 데이터 생성, 입력 처리, 정산 계산, CSV 내보내기
