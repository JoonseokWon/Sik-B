정산 조건

저장 상태 확인 중
회사 지급률과 멤버별 계약 지급률의 합계가 100%가 아니면 변경 내용이 저장되지 않습니다. 같은 날짜의 같은 시간대에 기준 횟수 이상 결제되면 승인이 필요합니다.

멤버별 정산

회사 분배금을 포함한 지급률 합계가 100%가 되도록 두고, 공통/개인 매출에 각 멤버의 계약 지급률을 적용합니다. 계약 조건은 계약서 참조에서 확인합니다.

매출 항목

공통 공연/콘서트는 공통 매출 풀로 따로 집계하고, 방송/라디오/개인 출연료는 개인 매출로 해당 멤버에게 귀속합니다.

식비 결제 건

식사 생략 또는 별도 식사 인원을 체크하면 해당 식비 배분에서 제외됩니다. 샐러드 주문 같은 예외는 특이 체크하고 사유와 실제 식비를 기록할 수 있습니다.

마케팅 사업부 급여 배부

직원별 월 급여를 전체 업무시간 중 아이돌 실제 투입시간 비율로 배부하고, 각 아이돌에게 정산할 금액에서 상계합니다.

더미데이터는 월 18일 × 8시간 = 144시간을 기준으로 합니다. 배부액 = 직원 월 급여 × 해당 아이돌 투입시간 ÷ 전체 업무시간. 비아이돌 업무시간에 해당하는 급여는 아이돌 정산에 배부하지 않습니다.
